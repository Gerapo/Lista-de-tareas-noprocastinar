# 📝 Lista de Tareas Avanzada

Proyecto desarrollado por Gerardo Polanco, estudiante de Ingeniería de Software, como parte del crecimiento profesional y técnico dentro del mundo del desarrollo web.  
Este proyecto tiene como objetivo aplicar conocimientos fundamentales de HTML, CSS y JavaScript, creando una aplicación funcional, clara y adaptable para cualquier usuario, añadiendo que puede ayudar a una persona que se le escapa las tareas, como yo.

---

## 🚀 ¿De qué trata?

Una **lista de tareas dinámica y práctica** donde se pueden:

- Agregar tareas con descripción, urgencia y fecha límite.
- Visualizar claramente las tareas pendientes y su prioridad.
- Marcar tareas como completadas y llevar un historial.
- Eliminar tareas completadas individualmente o con un clic.
- Adaptarse automáticamente a distintos tamaños de pantalla (diseño responsive).

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3** (con diseño adaptable)
- **JavaScript (Vanilla)** — sin frameworks
- **LocalStorage** para almacenamiento persistente

---

## 🧩 Funcionalidades clave

| Función                         | Descripción                                                                 |
|--------------------------------|-----------------------------------------------------------------------------|
| ✅ Agregar tarea               | Escribir tarea, elegir urgencia y fecha límite                             |
| 🔁 Responsive                 | Diseño se adapta perfectamente en celulares y pantallas pequeñas           |
| 📌 Niveles de urgencia        | Baja 🟢 | Media 🟠 | Alta 🔴                                             |
| ⏳ Fecha límite               | Visualización clara del vencimiento de tareas                               |
| ☑ Marcar como completada      | Al hacer clic en el texto, pasa al historial de finalizadas                |
| 🗑 Eliminar del historial      | Borrar individualmente o con botón “Limpiar todo”                          |
